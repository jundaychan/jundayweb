# 关于

## 我是谁

我是俊达（Junday），创业者，AI 实践者。

我不是 AI 工程师，也不是学术研究者。我是一个在一线做业务的人，每天都在用 AI 工具解决实际问题——写方案、做分析、出内容、管流程。

这个网站的所有内容都是我用 AI 写的。没错，包括你正在看的这段话。

## 这个网站是什么

**junday.net** 是一个面向中国企业创始人和老板的 AI 实战指南。

不讲论文，不吹概念，只回答一个问题：**AI 到底怎么帮我的生意赚钱或省钱？**

网站内容包括：
- **学 AI**：入门指南、核心概念、200+ 个 AI 概念速查
- **用 AI**：工具实测对比（真实价格和中国可用性）、提示词模板库（来自 GitHub 顶级开源项目）
- **AI 实战**：企业落地方法论、电商全链路 AI 方案、国内外真实案例
- **常见问题**：老板们真正关心的问题，直接回答

## 为什么做这个

因为我发现一个问题：**市面上的 AI 内容要么太技术，要么太忽悠。**

技术文章老板看不懂，营销号内容老板看完更焦虑。老板们需要的是：这个东西我今天能用吗？怎么用？花多少钱？有没有人已经用了？效果怎么样？

这个网站就是来回答这些问题的。

## 内容来源

我对内容质量有执念：**不编数据，不编案例，不编提示词。**

- 提示词模板来自 GitHub 高星开源项目（awesome-chatgpt-prompts 143K stars、DAIR.AI 72K stars 等）
- 案例数据来自企业公开报告和可信第三方（BCG、Klarna、Shopify 等）
- 工具信息来自各平台官网实测（2025 年数据）
- 法规信息来自官方文件和法院判例

如果某个信息我不确定，会标注"需验证"，而不是编一个看起来对的数字。
