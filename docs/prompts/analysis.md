# 数据分析提示词模板

> 不需要会Excel公式，不需要会SQL。把数据丢给AI，让它帮你看门道。

## 销售数据分析

**适用场景**：每月/每周的销售数据，需要快速发现问题和机会。

```
你是一个电商数据分析师。

请分析以下销售数据并给出经营建议：

【粘贴你的数据，可以是表格、截图中的数字、或文字描述，比如：
- 本月GMV：85万（上月92万，去年同期78万）
- 订单量：3200单（上月3500单）
- 客单价：265元（上月263元）
- 退货率：8.5%（上月6.2%）
- 流量：15万UV（上月18万UV）
- 转化率：2.1%（上月1.9%）】

请分析：
1. 关键指标变化总结（用红绿标注好坏，一目了然）
2. 问题诊断：GMV下降的主要原因是什么？（拆解到具体因素）
3. 亮点发现：哪些指标在变好？为什么？
4. 3条具体的改善建议（每条要可执行、有优先级）
5. 下月应该重点关注哪2-3个指标

要求：
- 不要只描述数据（"GMV下降了7.6%"我自己能看到），要给诊断和建议
- 建议要具体到可以马上做的动作
- 如果数据不够做判断，告诉我还需要什么数据
```

## 用户画像生成

**适用场景**：了解你的客户到底是谁，为选品和营销提供方向。

```
你是一个用户研究专家。

请根据以下信息帮我生成用户画像：
- 产品品类：【母婴用品，主打学步鞋】
- 价格区间：【100-200元】
- 主要销售平台：【淘宝+抖音】
- 已知用户信息：【80%女性下单、25-35岁为主、二三线城市居多】
- 用户好评中常提到的：【轻便、透气、孩子不排斥穿】
- 用户差评中常提到的：【鞋底不够软、码数偏小】

请输出：
1. 3个典型用户画像（每个包含：身份、年龄、城市、收入、核心焦虑、购买决策因素）
2. 用户购买旅程（从意识到需求→搜索→对比→下单→复购的完整链路）
3. 每个画像对应的营销策略建议
4. 基于差评的产品改进建议
```

## 市场调研报告

**适用场景**：进入新品类/新市场前的快速调研。

```
你是一个市场研究分析师。

请帮我做一份快速市场调研：
- 调研目标：【评估进入"宠物智能用品"赛道的可行性】
- 我的现有优势：【有电商运营经验、有供应链资源、团队5人】
- 预算：【首批备货+推广30万以内】
- 关注的问题：【市场有多大、竞争格局、利润空间、入局门槛】

请分析：
1. 市场规模和增长趋势（用可获得的公开数据）
2. 主要玩家和竞争格局（头部品牌、新锐品牌、白牌）
3. 消费者需求和痛点
4. 利润空间评估（典型产品的成本结构和毛利）
5. 入局建议：该不该做？如果做，从哪个细分品切入？
6. 风险提示

注意：如果你不确定具体数据，请标注"需验证"，不要编造数字。
```

## 竞品对比分析

**适用场景**：深度分析竞争对手，找到差异化方向。

```
你是一个竞争情报分析师。

请帮我做竞品对比分析：
- 我的产品：【描述你的产品、价格、定位】
- 竞品A：【名称、价格、主打卖点】
- 竞品B：【名称、价格、主打卖点】
- 竞品C：【名称、价格、主打卖点】

请输出：
1. 竞品对比矩阵（表格：维度 | 我 | 竞品A | 竞品B | 竞品C）
   维度包括：价格、核心卖点、目标人群、销量级别、视觉风格、营销策略
2. 每个竞品的优势和软肋
3. 市场空白点（他们都没做好的地方）
4. 我的差异化策略建议（3条，具体可执行）
5. 短期（1个月）和中期（3个月）行动建议
```

## 财务数据解读

**适用场景**：看不太懂财务报表，需要AI帮你翻译成人话。

```
你是一个中小企业财务顾问。

请帮我解读以下经营数据，用老板能听懂的话：

【粘贴你的数据，比如：
- 月收入：120万
- 成本：货品成本48万、人工25万、房租8万、推广费22万、物流12万
- 退款金额：9.6万
- 库存金额：65万】

请分析：
1. 利润结构拆解（表格：项目 | 金额 | 占比 | 是否健康）
2. 最大的"利润杀手"是哪几项
3. 哪些成本可以优化，怎么优化
4. 库存健康度评估（是否压了太多钱在库存上）
5. 和行业平均水平的对比（如果你知道的话）
6. 3条具体的降本增效建议

要求：不要用专业财务术语，用大白话。
```

## 行业趋势分析

**适用场景**：了解行业动态，辅助战略决策。

```
你是一个行业分析师。

请帮我分析以下行业的趋势：
- 行业：【跨境电商/宠物用品/知识付费/……】
- 我的关注点：【未来1-2年有哪些机会和风险】
- 我的身份：【中小企业老板，年营收500万-5000万】

请分析：
1. 行业近1-2年的关键变化（3-5个）
2. 正在崛起的新机会（要具体，不要"数字化转型"这种空话）
3. 即将到来的风险和挑战
4. AI对这个行业的具体影响（哪些环节会被AI改变）
5. 对我这种规模的企业，最值得关注的1-2个方向

注意：如果你对某些判断不确定，请标注"个人推测"和"建议进一步验证"。
```

## 定价策略分析

**适用场景**：产品定价拿不准，需要系统思考。

```
你是一个定价策略顾问。

请帮我分析定价策略：
- 产品：【描述产品】
- 成本：【单件成本XX元（含物料、人工、包装、物流）】
- 竞品价格：【竞品A XX元、竞品B XX元、竞品C XX元】
- 目标人群：【描述客户群体和消费能力】
- 销售渠道：【淘宝/抖音/私域/线下】
- 品牌定位：【高端/中端/性价比】

请给出：
1. 推荐定价区间和具体定价（附理由）
2. 不同价格点的利弊分析
3. 定价锚定策略（怎么让用户觉得这个价格合理）
4. SKU价格梯度设计（引流款/利润款/形象款）
5. 促销时的降价策略（日常价vs活动价的关系）
6. 涨价路径规划（如果未来想涨价，怎么做才不掉销量）
```
