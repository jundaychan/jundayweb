# 图片生成模型

> 电商需要大量产品图、海报、封面——AI 图片生成正在改变视觉内容的生产方式。

## 首选推荐

### Nano Banana（Google）

**推荐理由**：Google 最新的图片生成模型，集成在 Gemini 中，画质精细、文字渲染准确、支持图片编辑，是目前综合能力最强的图片生成工具之一。

| 优势 | 说明 |
|------|------|
| 画质精细 | 生成图片细节丰富，质感接近专业摄影 |
| 文字渲染 | 图片中的文字清晰准确，适合做海报和封面 |
| 图片编辑 | 不只能生成，还能修改已有图片（换背景、改细节） |
| 搜索增强 | 可以基于网络搜索结果生成更真实的图像 |
| 多风格 | 支持写实、插画、3D、扁平等多种风格 |

**适合场景**：

- **电商主图**：生成干净的产品展示图
- **课程封面**：专业感强的知识付费课程封面
- **社交媒体配图**：小红书、朋友圈的精美配图
- **海报设计**：促销海报、活动海报（文字渲染准确）
- **产品场景图**：将产品 P 到各种场景中

**使用方式**：通过 [Gemini](https://gemini.google.com) 网页版使用，在对话中直接描述你想要的图片

## 实战场景

### 电商产品图

```
提示词示例：
"一瓶精致的玻璃瓶护肤精华液，放在大理石台面上，
背景是柔和的自然光，旁边有几片绿叶装饰，
高端简约风格，白色背景，商业产品摄影风格"
```

::: tip 技巧
描述越具体越好：材质、光线、背景、风格、角度都要说清楚。先生成多张，选最好的再微调。
:::

### 知识付费课程封面

```
提示词示例：
"一个专业的课程封面设计，标题'AI时代的商业思维'，
深蓝色科技感背景，中间有抽象的AI芯片图形，
现代简约风格，16:9比例"
```

### 小红书配图

```
提示词示例：
"一张温馨的居家办公场景照片，MacBook放在木质桌面上，
旁边有一杯拿铁和一盆绿植，阳光从窗户洒进来，
Instagram风格，暖色调，生活感"
```

## 其他备选

### Midjourney

**艺术质量最高**的图片生成工具，特别适合品牌视觉、高端产品图。缺点是需要在 Discord 中使用，学习门槛稍高。

| 适合 | 不适合 |
|------|--------|
| 品牌视觉设计 | 快速批量出图 |
| 高端产品概念图 | 精确的文字渲染 |
| 创意海报 | 国内直接访问 |

### 即梦（字节跳动）

国内可直接使用，与抖音生态结合，适合生成短视频封面和社交媒体配图。免费额度较多。

### DALL·E 3（OpenAI）

集成在 ChatGPT 中，使用方便。对提示词理解准确，适合需要精确控制画面内容的场景。

### Stable Diffusion

开源免费，可本地部署。学习成本较高，但适合需要大批量生成或高度定制化的企业。

## 电商图片选择策略

::: tip 推荐工作流
1. **快速出图**：Nano Banana — 直接在 Gemini 中描述需求，秒出图
2. **高端品牌图**：Midjourney — 艺术质量最高，适合品牌主 KV
3. **国内快速使用**：即梦 — 免科学上网，免费额度多
4. **批量定制**：Stable Diffusion — 开源可控，适合有技术团队的企业
5. **精细编辑**：用 Nano Banana 生成后，在 Canva 中做最后的文字排版和微调
:::

::: warning 注意事项
- AI 生成的产品图建议用于参考和社交媒体，正式电商主图仍建议以实拍为主
- 注意 AI 生成图片的版权政策，各平台规定不同
- 如果图片中有人物，检查是否有不自然的细节（手指、文字等）
:::
