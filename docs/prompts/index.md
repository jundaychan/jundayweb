## 三个板块

### [商业决策角色](/prompts/business)

来自 ai-boost/awesome-prompts，每个都是**几千字的专业级系统提示词**。相当于给AI装上一个完整的商业大脑——包含方法论框架、决策流程、输出模板。

包含：产品经理、销售策略师、CFO财务顾问、增长黑客、市场调研总监、运营经理、数据分析师

**适合场景**：需要 AI 像一个资深顾问一样，系统性地帮你分析问题、给出方案。

### [万能角色扮演](/prompts/roles)

来自 awesome-chatgpt-prompts（全球 143K Star），精选了 20+ 个最适合企业使用的角色。每个提示词简短直接，复制粘贴就能用。

包含：广告策划、社交媒体经理、销售员、CEO、财务分析师、招聘顾问、创业点子生成器等

**适合场景**：快速让 AI 扮演某个角色，帮你完成具体任务。

### [提示词工程技巧](/prompts/techniques)

来自 DAIR.AI Prompt Engineering Guide（全球 72K Star），这是学术界和工程界公认的提示词方法论。

包含：零样本提示、少样本提示、思维链、思维树、自我一致性、ReAct、检索增强生成（RAG）等

**适合场景**：当你觉得 AI 的回答不够好，想系统性地提升提示词质量。

## 怎么用

### 商业角色提示词的用法

这些是**系统提示词**（System Prompt），设计上是作为 AI 的"人格设定"来用的：

1. 打开 ChatGPT / Claude
2. 复制整个角色提示词
3. 粘贴到对话开头（或设置为自定义指令 / System Prompt）
4. 然后像跟一个真正的顾问对话一样，提出你的具体问题

### 万能角色的用法

这些更轻量：

1. 复制提示词
2. 粘贴到 AI 对话框
3. 在后面接上你的具体需求
4. 直接发送

### 提示词技巧的用法

这些是**方法论**，不是模板：

1. 了解每种技巧的原理
2. 把技巧应用到你自己的提问方式中
3. 最实用的是**思维链**（Chain-of-Thought）：在提问末尾加一句"请一步一步思考"

::: tip 用哪个 AI 工具都行
这些提示词不限定工具。ChatGPT、Claude、Gemini、豆包、通义千问都能用。
:::
