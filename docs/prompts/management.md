# 经营管理提示词模板

> AI不只能写文案，还能帮你开会、管人、做计划。这些模板让AI成为你的管理助手。

## 会议纪要整理

**适用场景**：开完会一堆录音/笔记，需要快速整理成结构化纪要。

```
你是一个高效的行政助理。

请将以下会议内容整理为结构化的会议纪要：

【粘贴你的会议录音转文字/手写笔记】

输出格式：
1. 会议基本信息（时间、参会人、主题）
2. 核心结论（3-5条，每条一句话）
3. 待办事项清单（表格：事项 | 负责人 | 截止日期）
4. 分歧/未决问题（如有）
5. 下次会议安排

要求：
- 去掉废话和重复内容
- 待办事项必须有明确的负责人和截止日期
- 如果原文没有明确负责人，标注"待确认"
- 全文500字以内
```

## 周报/月报生成

**适用场景**：老板要求写周报但你不知道怎么写得又快又好看。

```
你是一个职场效率专家。

请帮我把以下工作内容整理成一份专业的周报：

本周做了什么：
【列出你这周的工作内容，可以是流水账】

输出格式：
1. 本周重点成果（3-5条，用数据说话）
2. 进行中的项目（进度百分比+下周计划）
3. 遇到的问题及需要的支持
4. 下周计划（3-5条）

要求：
- 用"完成了XX，带来了XX效果"的句式，不要用"做了XX"
- 数据量化：能用数字的地方一定用数字
- 问题部分要附带你的解决方案建议，不要只甩问题
- 全文300字以内
- 语气专业但不官僚
```

## 招聘JD撰写

**适用场景**：招人时写岗位描述，要吸引人又要准确筛选。

```
你是一个资深HR，精通人才市场和招聘文案。

请帮我写一份招聘JD：
- 岗位：【电商运营主管】
- 公司：【杭州，女装电商，年销售额5000万】
- 薪资范围：【15-25K/月】
- 汇报对象：【直接向创始人汇报】
- 核心要求：【3年以上电商运营经验，管理过3人以上团队，熟悉淘系+抖音】
- 加分项：【会用AI工具、有成功打爆款的案例】
- 公司亮点：【扁平管理、AI驱动、提供AI工具和培训】

要求：
1. 标题要吸引人（不要只是"招聘电商运营主管"）
2. 开头先卖公司（为什么要来这里而不是去别的公司）
3. 岗位职责5-7条（具体、可衡量）
4. 任职要求分"必须"和"加分"
5. 薪资福利要有诚意
6. 全文不超过500字
7. 语气真诚，不要"我们期待优秀的你"这种套话
```

## 面试问题设计

**适用场景**：面试候选人时，问出有质量的问题。

```
你是一个面试官培训师。

请帮我设计一套面试问题：
- 岗位：【AI运营专员】
- 核心能力要求：【会用AI工具、有电商运营基础、学习能力强、执行力强】
- 面试时长：【30分钟】
- 面试官：【创始人（非技术背景）】

请设计：
1. 破冰问题（1个，让候选人放松）
2. 经验验证问题（3个，判断是否真的会用AI工具）
3. 能力测试问题（2个，现场给一个任务看思路）
4. 价值观匹配问题（2个，看是否适合创业公司）
5. 反向提问引导（如何通过候选人的提问判断质量）

每个问题附带：
- 问题意图（你想通过这个问题看什么）
- 好回答的特征（什么样的回答说明这个人不错）
- 差回答的特征（什么样的回答是红旗）
```

## SOP编写

**适用场景**：把一个工作流程标准化，让新人也能照着做。

```
你是一个流程优化顾问。

请帮我把以下工作流程写成标准SOP：
- 流程名称：【电商新品上架流程】
- 涉及角色：【运营、设计、仓库】
- 当前做法：【运营确定产品→找设计做主图→写标题和描述→设置价格和SKU→上传到后台→仓库备货→上架推广】
- 当前问题：【经常漏步骤、新人不知道标准、各部门配合混乱】

输出格式：
1. 流程全景图（用文字描述步骤流转）
2. 每个步骤的详细说明（表格：步骤序号 | 负责人 | 具体动作 | 交付物 | 完成标准 | 预计耗时）
3. 关键检查点（哪些步骤容易出错，需要审核）
4. 异常处理（常见问题和解决方案）
5. 模板和工具清单（这个流程需要用到哪些文档/工具）
```

## 员工培训大纲

**适用场景**：需要培训团队使用AI工具或新技能。

```
你是一个企业培训设计师。

请帮我设计一个培训大纲：
- 培训主题：【电商团队AI工具使用培训】
- 培训对象：【运营、设计、客服，共15人，大部分没用过AI】
- 培训时长：【半天（4小时）】
- 目标：【培训结束后，每个人能用AI完成至少1个日常工作任务】
- 可用工具：【ChatGPT、AI生图工具】

请设计：
1. 时间安排表（每个环节的时间、内容、方式）
2. 开场破冰（消除"AI会替代我"的恐惧）
3. 分岗位的实操练习（运营/设计/客服各自练什么）
4. 现场作业（当场用AI完成一个真实任务）
5. 培训后跟进计划（怎么确保他们会持续用）
6. 需要提前准备什么（设备、账号、资料）
```

## 绩效考核方案

**适用场景**：AI时代，传统的"做图数量""接线量"考核已经过时了。

```
你是一个企业管理顾问，精通AI时代的绩效设计。

请帮我设计一套新的绩效考核方案：
- 岗位：【电商设计师】
- 团队规模：【3人】
- 现有考核方式：【按做图数量考核，月薪8000-12000】
- 问题：【引入AI后，做图数量意义不大，但不知道改成什么标准】

请设计：
1. 新的考核维度（3-5个，替代"做图数量"）
2. 每个维度的具体指标和权重
3. 评分标准（什么算优秀/合格/不合格）
4. 和薪资的挂钩方式
5. 过渡方案（从旧考核到新考核怎么平滑过渡）

参考原则：
- 考核结果（点击率、转化率）而非过程（做了多少张图）
- 鼓励AI提示词沉淀和团队知识共享
- 区分"执行型设计师"和"创意型设计师"
```
