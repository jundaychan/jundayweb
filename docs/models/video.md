# 视频生成模型

> 短视频时代，视频内容是电商和知识付费的核心载体。AI 正在让视频制作从"天"缩短到"分钟"。

## 首选推荐

### Seedance 2.0（字节跳动）

**推荐理由**：字节跳动推出的最新视频生成模型，在运动质量、物理真实感和指令遵循方面表现突出，特别适合电商和短视频场景。

| 优势 | 说明 |
|------|------|
| 运动自然 | 人物动作和物体运动极为流畅自然 |
| 物理真实 | 符合物理规律，液体、布料、光影效果逼真 |
| 指令精准 | 能准确理解复杂的文字描述，生成匹配的视频 |
| 电商适配 | 背靠字节，天然适配抖音电商的内容需求 |
| 图生视频 | 支持从静态图片生成动态视频 |

### 适合场景

#### 电商
- **商品展示视频**：上传产品图，自动生成 360° 展示动画
- **使用场景视频**：将产品 P 到真实使用场景中并添加动态效果
- **直播切片素材**：快速生成商品介绍短视频
- **节日营销视频**：快速产出节日主题的促销短视频

#### 知识付费
- **课程宣传片**：用文字描述自动生成课程推广视频
- **知识短视频**：将知识点可视化为短视频内容
- **学员案例展示**：生成学员成长故事的视觉化内容
- **社交媒体引流**：批量生产吸引眼球的短视频

### 使用方式

1. 访问即梦（Seedance 集成在字节的即梦平台中）
2. 输入文字描述或上传参考图片
3. 选择视频参数（时长、比例、风格）
4. 等待生成（通常 1-3 分钟）
5. 下载后用剪映做最终剪辑

## 电商视频生产流程

```
第一步：策划
├── 确定视频目的（展示/种草/促销）
├── 写好脚本和分镜描述
└── 准备产品图片素材

第二步：AI 生成
├── 用 Seedance 2.0 生成主体视频片段
├── 用 Nano Banana 生成需要的静态画面
└── 用 Suno 生成背景音乐

第三步：后期合成
├── 用剪映拼接视频片段
├── 添加字幕和文字标注
├── 叠加背景音乐和配音（MiniMax）
└── 导出发布

第四步：分发
├── 抖音/快手（竖版 9:16）
├── 小红书（1:1 或 3:4）
├── 视频号（横版或竖版）
└── 商品详情页（横版 16:9）
```

## 注意事项

::: warning 当前限制
- **时长有限**：目前大多数模型单次生成 5-15 秒，长视频需要拼接
- **细节控制**：AI 视频的精确控制不如传统视频制作，适合氛围类内容
- **一致性**：多段视频之间的人物、场景一致性仍是挑战
- **平台政策**：部分平台对 AI 生成视频有标注要求，需关注
:::

::: tip 务实的建议
AI 视频目前最适合做**短视频素材**和**辅助画面**，而不是完全替代传统视频拍摄。最高效的方式是 AI 生成 + 真人出镜相结合。
:::
