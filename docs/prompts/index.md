# 提示词模板库

> 别学理论了，直接抄作业。每个模板都是实战验证过的，复制粘贴就能用。

## 为什么做这个

老板们最常跟我说的一句话是：**"我知道AI有用，但我不知道怎么问它。"**

提示词（Prompt）就是你给AI下的指令。同样是ChatGPT，你问"帮我写个产品描述"和你用一个精心设计的模板问，出来的东西天差地别。

这个模板库就是帮你解决"不知道怎么问"的问题——**直接复制，改几个关键词，就能用。**

## 怎么用这个模板库

1. **找到你的场景**：按下面的分类找到最接近你需求的模板
2. **复制模板**：每个模板都有一个代码框，直接复制
3. **替换变量**：把 `【】` 里的内容换成你自己的信息
4. **粘贴到AI**：打开ChatGPT、Claude或任何AI工具，粘贴进去
5. **根据结果微调**：AI的第一次输出不满意？直接说"更口语化一点"或"再简短一些"

::: tip 用哪个AI工具都行
这些模板不限定工具。ChatGPT、Claude、Gemini、豆包、通义千问都能用。如果不确定用哪个，推荐先试ChatGPT或Claude。
:::

## 使用技巧

### 让AI输出更好的3个诀窍

**1. 给角色**：告诉AI"你是一个10年经验的电商运营专家"，比直接问效果好得多。

**2. 给约束**：不要让AI自由发挥。"用200字以内""列5个要点""用口语化的语气"——越具体，输出越可控。

**3. 给例子**：如果你有一个满意的文案，贴给AI说"参考这个风格写"，比你花30分钟描述风格更高效。

### 效果不好怎么办？

不要重新开始，直接追问：
- "更口语化一点，少用成语"
- "太长了，压缩到100字"
- "语气再轻松一些，像朋友聊天"
- "加入更多具体数据"
- "换一个完全不同的角度重写"

AI是可以"调教"的，几轮对话之后就能找到你要的感觉。
