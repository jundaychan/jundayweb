# 电商运营提示词模板

> 从选品到售后，电商全链路的AI提示词。复制 → 替换【】里的内容 → 粘贴到AI → 直接用。

## 产品标题优化

**适用场景**：淘宝/天猫/拼多多/抖音商品标题，需要覆盖搜索关键词又要吸引点击。

```
你是一个10年经验的电商运营专家，精通平台搜索算法和用户心理。

请帮我优化以下商品的标题：
- 商品：【白色羽绒服女中长款】
- 平台：【淘宝】
- 目标人群：【25-35岁女性，注重性价比】
- 核心卖点：【90%白鹅绒、防风防水、可拆卸帽子】
- 价格区间：【399-499元】

要求：
1. 生成5个不同版本的标题
2. 每个标题控制在30个字以内
3. 包含3-5个高搜索量关键词
4. 前10个字要有吸引力（用户只看到这么多）
5. 标注每个标题的侧重点（搜索流量型/点击率型/转化型）
```

## 爆款产品描述

**适用场景**：产品详情页的核心文案，需要既有卖点又有说服力。

```
你是一个顶级电商文案写手，擅长用"痛点+方案+证据"的结构写出高转化文案。

请为以下产品写一份详情页文案：
- 产品：【儿童学步鞋】
- 价格：【129元】
- 核心卖点：【1/3处科学弯折设计、360°防撞鞋头、真皮透气】
- 目标用户：【新手妈妈，孩子8-18个月】
- 竞品对比优势：【比泰兰尼斯便宜一半，但用了同等级的材料】
- 用户最大顾虑：【怕买错鞋影响孩子脚部发育】

要求：
1. 开头先戳痛点（不超过2句话）
2. 中间用"场景化"描述卖点（用户能想象自己在用）
3. 加入信任背书（材料、检测、对比数据）
4. 结尾制造紧迫感
5. 全文800字以内，用短句，不要用"的的的"
6. 不要用"绝绝子""yyds"等网络用语
```

## 主图卖点文案

**适用场景**：电商主图上的短文案，需要3秒内抓住注意力。

```
你是一个精通视觉营销的电商设计主管。

请为以下产品设计5组主图文案（主图上的大字+小字组合）：
- 产品：【便携式筋膜枪】
- 核心卖点：【仅重320g、45dB超静音、Type-C充电、续航8小时】
- 价格：【199元】
- 目标人群：【办公室久坐白领、健身爱好者】

要求：
1. 每组包含：大标题（6字以内）+ 副标题（12字以内）
2. 大标题要有冲击力，副标题补充关键信息
3. 至少2组用数据说话
4. 至少1组走情感路线
5. 语言简洁有力，不要废话

示例格式：
大标题：轻到忘记它存在
副标题：320g极轻机身 办公室随时用
```

## 智能客服话术库

**适用场景**：设置AI客服或培训人工客服时，生成标准回复话术。

```
你是一个电商客服培训师，需要帮我设计一套客服话术库。

店铺信息：
- 品类：【女装】
- 价格区间：【200-500元】
- 发货时间：【48小时内】
- 退换货政策：【7天无理由退换，运费险覆盖】

请针对以下10个常见场景，每个写2个版本的回复（一个正式版、一个亲切版）：
1. 问尺码怎么选
2. 问什么时候发货
3. 催发货
4. 问面料材质
5. 讲价/要优惠
6. 问能不能退换
7. 收到货不满意
8. 问搭配建议
9. 问是否正品
10. 大促期间问什么时候到货

要求：
- 每条回复控制在50字以内
- 语气友好但不油腻（不要"亲亲""宝子"）
- 关键信息加【】标注（方便客服快速替换）
- 需要转人工的场景要明确标注
```

## 差评回复模板

**适用场景**：回复差评，既要解决问题又要让围观者看到态度。

```
你是一个资深电商客户关系经理。

请帮我回复以下差评：
- 差评内容：【"质量太差了，穿了一次就起球，而且发货还慢，等了5天"】
- 产品：【羊毛混纺毛衣，售价299元】
- 实际情况：【发货确实比平时慢了2天（大促期间），起球问题可能是洗涤方式不对】

要求：
1. 先道歉（不要辩解）
2. 针对每个具体问题给出解决方案
3. 提供补偿选项（不要直接说退款，给选择）
4. 语气真诚、不卑不亢
5. 控制在150字以内
6. 要让其他看评价的人觉得"这家店态度不错"
7. 不要用模板感很重的套话
```

## 竞品分析

**适用场景**：快速了解竞争对手的策略，找到自己的差异化方向。

```
你是一个电商行业分析师。

请帮我分析以下竞品并给出策略建议：
- 我的产品：【儿童防晒衣，售价89元，UPF50+，速干面料】
- 竞品1：【蕉下儿童防晒衣，168元，主打品牌溢价和高级感】
- 竞品2：【安踏儿童防晒衣，129元，主打运动专业】
- 竞品3：【某白牌，49元，低价走量】
- 我的优势：【面料和蕉下同厂，但价格只有一半】

请分析：
1. 每个竞品的核心定位和目标人群（用表格对比）
2. 每个竞品的主图/详情页策略特点
3. 我的差异化机会在哪里
4. 给出3个具体的卖点包装建议
5. 推荐2个可以切入的细分人群方向
```

## SKU裂变策划

**适用场景**：同一个产品通过不同卖点包装，裂变出多个链接覆盖不同人群。

```
你是一个精通电商搜索和人群运营的操盘手。

我想用"一品多链接"策略，把同一款产品裂变成不同链接切入不同人群。

产品信息：
- 产品：【折叠轮椅】
- 价格区间：【800-1200元】
- 核心功能：【可折叠、避震、防后溜、人体工学头枕】
- 当前主打卖点：【轻便折叠，仅重12kg】

请帮我设计5个裂变方向，每个方向包含：
1. 目标人群画像（谁买、为谁买、什么场景下买）
2. 主打卖点（只突出1个核心卖点）
3. 标题方向（15字以内的主图大标题）
4. 主图视觉策略（拍什么场景）
5. 搜索关键词方向（3-5个长尾词）
```

## 详情页卖点排序

**适用场景**：详情页图片很多但逻辑混乱，需要重新梳理说服链路。

```
你是一个转化率优化专家。

请帮我重新规划详情页的内容排序：
- 产品：【家用空气炸锅，售价259元】
- 目标用户：【25-40岁，租房或小户型，不太会做饭】
- 竞品最常见的详情页结构：【品牌故事→功能参数→使用场景→用户评价】
- 我们的核心卖点：【不用翻面就能均匀加热、5分钟快速预热、可视窗口、可拆洗】

请给出：
1. 推荐的详情页内容排序（10个版块，写明每个版块的目的）
2. 每个版块的核心文案方向（一句话）
3. 前3屏的文案（用户平均只看前3屏就决定买不买）
4. 哪些信息可以不放（很多详情页信息冗余）
```

## 直播话术脚本

**适用场景**：直播带货的话术框架，适合主播和数字人直播。

```
你是一个直播带货话术编剧。

请为以下产品写一套3分钟的直播话术：
- 产品：【酵素洗衣液，1kg装，售价29.9元，买2送1】
- 核心卖点：【生物酶去渍、不伤手、自然花香不刺鼻】
- 用户痛点：【传统洗衣液洗不干净领口袖口、味道太化工】
- 目标：【引导下单，强调限时优惠】

话术结构要求：
1. 开场钩子（10秒，制造好奇或共鸣）
2. 痛点放大（20秒，让观众觉得"说的就是我"）
3. 产品展示（60秒，边说边演示）
4. 信任背书（20秒，成分/检测/销量/老客反馈）
5. 价格锚定（20秒，对比原价、竞品价）
6. 逼单话术（30秒，限量/倒计时/加赠）

语气要求：像朋友推荐，不要像电视购物喊叫。
```
