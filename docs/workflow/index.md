# 工作流自动化：让 AI 帮你干重复的活

> 你每天花 2 小时整理数据、发报表、跟进客户——这些活可以让 AI 自动干。

## 什么是 AI 工作流

**工作流**（Workflow）= 把一系列步骤串起来自动执行。加上 AI 节点，就是 AI 工作流。

```
例子：每日销售报告自动化

以前（手动）：
  登录后台 → 导出数据 → 用 Excel 分析 → 写总结 → 发到群里
  耗时：30-60 分钟/天

现在（AI 工作流）：
  定时触发 → 自动拉数据 → AI 分析趋势 → AI 写总结 → 自动发群
  耗时：0 分钟（全自动）
```

### 工作流 vs Agent：什么时候用哪个

这是一个关键区分，选错了会浪费时间：

| | 工作流 | Agent |
|--|--------|-------|
| **步骤** | 固定的，你预先设计好 | AI 自己决定下一步 |
| **可预测性** | 高，每次执行结果一致 | 低，AI 可能走不同路径 |
| **适合** | 重复性任务、流程明确 | 复杂决策、开放式任务 |
| **出错风险** | 低 | 较高（AI 可能判断失误） |
| **调试难度** | 容易（每步有日志） | 较难（AI 思路难追踪） |

**Anthropic 的建议**：

> "从工作流开始，不要急着上 Agent。先用固定步骤把流程跑通，效果稳定了再考虑让 AI 自主决策。"
>
> — [Building Effective Agents](https://www.anthropic.com/research/building-effective-agents)

### 8 个最值得自动化的场景

| 场景 | 手动耗时 | 自动化后 | 推荐平台 |
|------|---------|---------|---------|
| **每日销售报告** | 30 分钟/天 | 全自动 | n8n / 飞书多维表格 |
| **客户跟进提醒** | 每天检查 CRM | 自动提醒 | n8n / 钉钉宜搭 |
| **社交媒体发布** | 手动写+发 | AI 写+定时发 | n8n / Coze |
| **竞品监控** | 手动浏览网站 | 自动抓取+AI 分析 | n8n |
| **发票/订单处理** | 手动录入 | AI 识别+自动录入 | Dify 工作流 |
| **邮件自动回复** | 逐封处理 | AI 分类+模板回复 | n8n / 飞书 |
| **周报生成** | 1-2 小时/周 | AI 汇总+自动生成 | n8n / Dify |
| **库存预警** | 手动查库存 | 低于阈值自动通知 | n8n / 钉钉宜搭 |

## 平台选择

### 开源/国际平台

| 平台 | 特点 | 适合谁 | 成本 |
|------|------|--------|------|
| **[n8n](https://n8n.io)** | 700+ 集成，可视化拖拽，开源可自部署 | 想要灵活定制的技术团队 | 开源免费 / 云版 €20/月起 |
| **[Dify](https://dify.ai) 工作流** | RAG + Agent + 工作流一站式 | 已在用 Dify 做 RAG/Agent 的团队 | 开源免费 |
| **[Coze](https://www.coze.cn) 工作流** | 零代码，字节出品，和豆包/飞书生态打通 | 不会代码、在字节生态内 | 免费 |

### 中国本土平台

| 平台 | 特点 | 适合谁 | 成本 |
|------|------|--------|------|
| **飞书多维表格** | 内置 AI 公式，自动化流程，和飞书深度集成 | 用飞书的团队 | 飞书企业版含 |
| **钉钉宜搭** | 低代码搭建，和钉钉/阿里云生态打通 | 用钉钉的团队 | 钉钉企业版含 |
| **腾讯微搭** | 和企业微信/腾讯云打通 | 用企业微信的团队 | 按量付费 |

### 怎么选

```
已经用飞书 → 飞书多维表格（零学习成本）
已经用钉钉 → 钉钉宜搭（零学习成本）
要跨平台对接 → n8n（700+ 集成最灵活）
要 RAG + 工作流 → Dify（一站式）
零代码快速搭 → Coze 工作流
```
