# 企业应用场景

> 不知道从哪里开始用 AI？按部门找你最痛的场景。

## 优先级判断公式

```
AI化优先级 = 重复频率 × 每次耗时 × 容错程度
```

翻译成人话：**每天都在做、每次花很久、允许不完美的工作，最适合让 AI 做。**

## 从哪里开始

不要一口气全上。按这个顺序来：

1. **本周**：老板自己先用 AI 写一封邮件、做一个方案 → [怎么开始](/guide/getting-started)
2. **下周**：找到团队最痛的一个重复性工作，用 AI 试试
3. **第三周**：让 2-3 个愿意尝试的员工一起用
4. **一个月后**：评估效果，决定是否扩大
