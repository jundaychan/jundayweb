# 会动手的 AI：执行型 Agent 工具指南

> 普通聊天 AI 只会"动嘴"给你建议；这一类 **执行型 Agent** 会"动手"——直接在你电脑上改文件、写代码、做 PPT、跑任务。它们不是助手，是**会干活的数字员工**。

这页讲三个最值得老板关注的执行型 Agent，按"中国能不能直接用"排序：

| 工具 | 出品方 | 擅长 | 中国直接用 | 上手门槛 |
|------|--------|------|:----:|------|
| **腾讯 WorkBuddy** | 腾讯云 | 办公（PPT/Word/Excel、文件批处理） | ✅ | 低，下载即用 |
| **Codex** | OpenAI | 写代码、技术任务 | ❌ 需翻墙 | 中 |
| **Claude Code** | Anthropic | 写代码、技术任务（公认最强） | ❌ 需翻墙 | 中 |

::: tip 老板视角的一句话
**不懂技术、想让 AI 帮你处理日常办公** → 直接看 WorkBuddy。
**你或团队有人写代码 / 想做点小工具** → 看 Codex 和 Claude Code。
:::

## 一、腾讯 WorkBuddy：国内能直接用的"数字同事"

**是什么**：腾讯云推出的桌面级 AI Agent。你用一句大白话描述任务，它自己拆解步骤、读取你本地文件夹里的资料、自动干活。

**能帮老板干什么**：

- **批量处理文件**：把一个文件夹里的几十份资料，自动整理成 Excel 表 / 汇总报告
- **自动生成办公成果**：PPT 演示文稿、Word 文档、Excel 表格、PDF，一句话生成
- **远程指挥电脑**：配置好企业微信或 QQ 后，**人在外面用手机发条消息，就能让办公室的电脑帮你跑任务**
- **接入你常用的工具**：微信、飞书、钉钉、企业微信、QQ 都能连，还支持 MCP 协议和 Skills 扩展

**为什么值得重点看**：这是少有的**国内厂商出品、国内直接可用、不用翻墙**的执行型 Agent。对不懂技术的老板和团队，这是上手门槛最低的一个。

**怎么上手**：去腾讯云官网下载 Windows / macOS 安装包，按向导装好，首次启动授权访问文件夹即可。

> 📖 **看完整指南** → [腾讯 WorkBuddy 完全上手指南](/tools/workbuddy-guide)（安装、接入微信/飞书/钉钉、典型场景、安全提醒）
> 原始出处：[WorkBuddy 使用指南（腾讯云开发者社区）](https://cloud.tencent.com/developer/article/2638410)

::: warning 安全提醒
执行型 Agent 能读写你的本地文件、操作电脑。授权访问的文件夹**只给它该看的**，别一上来就把整个 D 盘的资料都开放。重要数据先在隔离的测试文件夹里跑通再说。
:::

## 二、OpenAI Codex：把写代码这件事"外包"给 AI

**是什么**：OpenAI 出品的编程 Agent。它不是聊天框，而是一个能**真正读你的项目、自己写代码、运行测试、改 bug** 的执行型工具，能在终端、编辑器和云端运行。

**能帮老板干什么**：

- 把"我想要一个能自动统计每天订单的小工具"这种需求，**从想法变成能跑的程序**
- 改网站、做小程序原型、写自动化脚本——以前要外包给程序员的小活，自己也能试
- 一个有意思的观点：**编程小白可能比老程序员更受益**——因为你没有"必须自己写"的包袱，更愿意把活完全交给 AI

**中国可用性**：需要 ChatGPT 付费账号 + 翻墙，存在合规风险（用 VPN 访问违反平台条款）。

**怎么上手**：

## 三、Claude Code：技术圈公认最强的编程 Agent

**是什么**：Anthropic（Claude 母公司）出品的终端编程 Agent，由 Claude 模型驱动。它能理解你**整个代码库**，跨多个文件干活，执行写功能、修 bug、处理 Git 等工程任务。在程序员群体里，它被普遍认为是当前编程能力最强的 AI 工具之一。

**能帮老板干什么**：和 Codex 类似——把技术需求交给 AI 执行。如果你团队里有技术人员，这是提效最猛的工具之一；Anthropic 自己内部团队大量用它来开发。

**几个实用心法**（来自官方最佳实践）：

- **先规划、再动手**：用 Plan 模式让它先想清楚方案，确认后再让它写——避免它一头扎进错误方向
- **管好上下文**：长对话用 `/clear` 重置、`/compact` 压缩，别让它"记太多反而糊涂"
- **给它一份"项目说明书"**：在项目里放一个 `CLAUDE.md`，写清项目背景和规矩，它每次都会读

**中国可用性**：和 Codex 一样，国内无法直接访问，需翻墙 + 海外账号，有合规风险。

**怎么上手**：

## 怎么选：一张决策图

```
你想让 AI 帮你干什么？
│
├─ 处理办公文档 / 不懂技术 / 不想翻墙
│     └─→ 腾讯 WorkBuddy（国内直接用）
│
└─ 写代码 / 做小工具 / 技术团队提效
      ├─ 想要最友好的中文入门 → Codex（看 MaynorAI 教程）
      └─ 想要最强能力、不在乎折腾 → Claude Code
```

::: tip 共同的正确用法
不管哪个工具，**从一个低风险小场景开始**：挑一件做错了也没损失的小事，让 AI 完整跑一遍，你在旁边看它怎么干。跑通了、信得过了，再交给它更重要的活。这和我们在 [Agent 篇](/ai-landing/agents) 里反复强调的"从简单开始、不要跳步"是一个道理。
:::
