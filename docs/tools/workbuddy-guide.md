# 腾讯 WorkBuddy 完全上手指南：国内能直接用的"数字同事"

> WorkBuddy 是腾讯云出品的桌面级 AI Agent。你用一句大白话下指令，它就能读你电脑里的文件、自动干活、产出 PPT/Word/Excel。**国内直接可用、不用翻墙**——是三个执行型 Agent 里对老板和非技术团队最友好的一个。
>
> 参考：[WorkBuddy 使用指南（腾讯云开发者社区）](https://cloud.tencent.com/developer/article/2638410)

## 一、它把 AI 从"顾问"变成了"员工"

普通 AI 给你的是建议（什么该做），WorkBuddy 给你的是动作（直接帮你做）。它的核心是：**听懂你一句话的意图 → 拆成可执行的步骤 → 读取本地资料 → 自己干完 → 交出成果。**

![WorkBuddy：从一句话到办公成果](/images/guides/workbuddy-flow.svg)

它能干的活，都是老板日常最头疼的重复劳动：

- **批量处理文件**：把一个文件夹里几十份资料，自动整理成一张 Excel 或一份汇总报告
- **一键出办公成果**：PPT 演示文稿、Word 文档、Excel 表格、PDF，描述清楚就能生成
- **手机远程指挥电脑**：配好企业微信或 QQ 后，人在外面发条消息，办公室的电脑就替你把活跑了
- **接你常用的工具**：微信、企业微信、飞书、钉钉、QQ 都能连，还支持 MCP 协议和 Skills 扩展

## 二、怎么安装（三步搞定）

1. 去腾讯云官网下载 WorkBuddy 安装包（Windows / macOS 都有），按向导装好
2. 首次启动，**授权它访问指定的文件夹**（这一步决定它能"看到"哪些资料）
3. 装完即可用——这是它比 Codex / Claude Code 省心的地方：不用配环境、不用 API key、不用翻墙

## 三、接入你的办公工具（手机遥控的关键）

WorkBuddy 真正好用的地方，是把它接到你天天用的沟通工具上。各平台思路：

![WorkBuddy 接入五大办公工具](/images/guides/workbuddy-connect.svg)

| 平台 | 大致接入方式 |
|------|-------------|
| **微信 / 微信客服号** | 通过客服号集成，对话即可下指令 |
| **企业微信** | API 模式接入，适合团队共享 |
| **飞书** | 创建应用、配置权限后接入 |
| **钉钉** | 配置应用与机器人 |
| **QQ** | 通过开放平台接入 |

接好之后，你就能**用手机给电脑发任务**——这是 WorkBuddy 最有"数字同事"感的用法。具体每个平台的配置步骤，跟着 [腾讯云的图文指南](https://cloud.tencent.com/developer/article/2638410) 走一遍即可。

## 四、典型场景（老板会用上的）

- "把'2026销售'这个文件夹里所有的日报，汇总成一份月度 Excel 表" → 它读文件、提数据、出表格
- "根据这份产品资料，做一个 10 页的招商 PPT" → 它直接生成可编辑的 PPT
- "把这堆 Word 合同里的关键条款提取出来，列成清单" → 批量读取 + 结构化整理
- 出差路上用手机发一句："把今天的会议纪要整理好发我邮箱" → 电脑端自动执行

## 五、安全提醒（执行型 Agent 通用铁律）

WorkBuddy 能读写你的本地文件、操作电脑，所以：

::: warning 三条安全底线
1. **只授权该看的文件夹**，别一上来把整个 D 盘都开放给它
2. **重要 / 敏感数据**先在隔离的测试文件夹里跑通，确认它行为靠谱再用于真实业务
3. **远程遥控功能**要管好你的微信 / QQ 账号安全，别让别人借你的账号发指令
:::
