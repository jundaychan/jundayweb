# AI 编程工具

> 当前最热门的 AI 编程工具，了解它们各自的特点，选择最适合你的。

## 27. GitHub Copilot

微软/GitHub 推出的 AI 编程助手，集成在编辑器中，能根据注释和上下文实时补全和生成代码。

## 28. Cursor

基于 VS Code 的 AI 原生代码编辑器，深度集成 AI 能力，支持对话式编程、代码修改和整个项目的理解。

## 29. Claude Code

Anthropic 推出的命令行 AI 编程工具，能直接在终端中理解整个项目并执行复杂的编码任务。

## 30. Windsurf（原 Codeium）

AI 驱动的代码编辑器和编程助手，提供智能代码补全和对话式开发体验。

## 31. VS Code（Visual Studio Code）

微软推出的免费代码编辑器，是目前最流行的开发工具，几乎所有 AI 编程插件都支持它。

## 32. Bolt.new

浏览器中直接使用 AI 生成、运行和部署全栈 Web 应用的平台，无需本地配置开发环境。

## 33. Lovable（原 GPT Engineer）

通过自然语言描述就能让 AI 生成完整 Web 应用的工具，特别适合快速原型验证。

## 34. v0 by Vercel

Vercel 推出的 AI UI 生成工具，用自然语言描述界面需求就能生成可用的前端组件代码。

## 35. Replit

在线集成开发环境，内置 AI 助手，支持在浏览器中编写、运行和部署代码，降低开发入门门槛。

## 36. ChatGPT / Claude / Gemini

通用 AI 对话工具，可以用来解释代码、调试错误、学习编程概念、生成代码片段。

## 37. GitHub

全球最大的代码托管和协作平台，是开发者存放代码、协同工作和使用开源项目的核心基础设施。
