# 编程基础概念

> 不需要你会写代码，但理解这些基础概念能让你和技术团队更高效地沟通。

## 1. 代码（Code）

用人类可读的文字告诉计算机要做什么的指令集合，是所有软件和 AI 应用的基础。

## 2. 编程语言（Programming Language）

人与计算机沟通的"语言"，常见的有 Python、JavaScript、TypeScript 等，其中 Python 是 AI 领域最主流的语言。

## 3. Python

目前 AI 和数据科学领域使用最广泛的编程语言，语法简洁易学，拥有丰富的 AI 相关库和工具。

## 4. JavaScript / TypeScript

网页和应用开发的主流语言，TypeScript 是 JavaScript 的增强版，适合构建 AI 驱动的 Web 应用和前端界面。

## 5. API（应用程序编程接口，Application Programming Interface）

不同软件之间"对话"的标准接口，比如你的应用通过 API 调用 OpenAI 的模型来获得 AI 能力。

## 6. SDK（软件开发工具包，Software Development Kit）

开发者用来快速接入某项服务的工具包，比如 OpenAI SDK 让你几行代码就能调用 GPT 模型。

## 7. JSON（JavaScript Object Notation）

一种轻量级的数据格式，AI 应用中几乎所有的 API 请求和响应都使用这种格式来传递信息。

## 8. 变量与数据类型（Variables & Data Types）

程序中存储和处理信息的基本方式，就像 Excel 中的单元格可以存放数字、文字或日期一样。

## 9. 函数 / 方法（Function / Method）

一段可以反复调用的代码块，把常用操作封装起来，避免重复编写相同的代码。

## 10. 数据库（Database）

结构化存储和管理数据的系统，AI 应用需要数据库来保存用户信息、对话历史、业务数据等。

## 11. 前端与后端（Frontend & Backend）

前端是用户看到和交互的界面，后端是处理逻辑和数据的服务器端，AI 功能通常在后端实现。

## 12. 开源（Open Source）

代码公开、任何人都可以免费使用和修改的软件，AI 领域大量核心工具和模型都是开源的。

## 13. 命令行 / 终端（Command Line / Terminal）

通过输入文字命令来操控计算机的界面，很多 AI 工具和开发操作都需要在终端中完成。

## 14. Markdown

一种简单的文本格式标记语言，用于编写文档、README 说明、AI 对话中的格式化内容。
