# 营销文案提示词模板

> 让AI帮你写出不像AI写的文案。关键是给足上下文、给好约束、给对角色。

## 小红书种草文

**适用场景**：发小红书推广产品，需要"真实感"而非"广告感"。

```
你是一个小红书博主，粉丝3万，风格是"真实分享、不吹不黑"。

请帮我写一篇小红书种草笔记：
- 产品：【某品牌玻尿酸保湿面霜，128元/50ml】
- 我的真实使用感受：【用了两周，干皮不紧绷了，但不算惊艳，性价比不错】
- 产品亮点：【玻尿酸+神经酰胺双修护、无香精色素、国货】
- 目标读者：【20-30岁，干皮/混干，预算200以内】

写作要求：
1. 标题要有"点击欲"，5个备选
2. 开头用"个人经历"切入，不要上来就介绍产品
3. 中间穿插真实使用细节（质地、上脸感受、持妆效果）
4. 要有一个"小缺点"（增加真实感）
5. 结尾自然带出产品，不要硬广
6. 全文300-500字
7. 加上合适的emoji，但不要每句都加
8. 禁止使用：绝绝子、yyds、真的会谢、无限回购
```

## 朋友圈营销文案

**适用场景**：发朋友圈推广产品/服务，要像"分享"不像"广告"。

```
你是一个私域运营专家。

请帮我写5条朋友圈文案：
- 产品/服务：【AI商业咨询，帮企业老板梳理AI落地方案】
- 目标读者：【企业老板、创始人、高管】
- 我的身份：【创业者，自己用AI做业务的实践者】
- 想传达的：【我真的懂AI+商业，不是理论派】

要求：
1. 每条100字以内
2. 5种不同风格：故事型、数据型、提问型、干货型、生活型
3. 不要出现"欢迎咨询""私我""戳我"等硬广词
4. 要像在分享一个发现，不像在卖东西
5. 让人看完想点赞或评论

示例风格参考：
"今天帮一个做童装的朋友看了下他的AI出图流程，发现他一张主图花80块找外包做，其实AI 3分钟就搞定了。省下来的钱够他多打两天直通车。"
```

## 短视频脚本

**适用场景**：抖音/视频号的短视频脚本，需要前3秒抓人。

```
你是一个短视频编导，擅长"钩子开头+干货中间+行动结尾"的结构。

请帮我写一个60秒的短视频脚本：
- 主题：【为什么你的AI工具买了没人用】
- 目标观众：【企业老板】
- 核心观点：【93%的预算花在工具上，7%花在培训上，所以失败】
- 我的身份：【AI商业顾问】
- 视频风格：【真人出镜口播，节奏快】

脚本格式要求：
1. 前3秒的钩子（必须让人停住不划走）
2. 标注每段的时间和画面建议
3. 语言要口语化，像在和朋友聊天
4. 结尾要有互动引导（但不要"点赞关注"这种老套的）
5. 全程不超过250字（60秒的口播量）
```

## 促销活动策划

**适用场景**：大促/日常活动的完整方案。

```
你是一个电商活动策划专家。

请帮我策划一个促销活动：
- 店铺品类：【女装，客单价200-500元】
- 活动时间：【7天】
- 目标：【清库存+拉新客，目标GMV 50万】
- 预算：【5万元（含优惠券补贴和推广费）】
- 当前粉丝量：【10万】
- 库存压力最大的品：【春季连衣裙，3000件】

请给出：
1. 活动主题（要有记忆点，不要"年中大促"这种通用的）
2. 优惠机制设计（阶梯满减/限时折扣/赠品策略）
3. 7天节奏规划（预热→爆发→返场的具体安排）
4. 推广渠道和预算分配
5. 关键数据指标和监控节点
6. 风险预案（卖太快怎么办/卖不动怎么办）
```

## 品牌故事

**适用场景**：官网、详情页、融资PPT中的品牌介绍。

```
你是一个品牌策划顾问。

请帮我写一个品牌故事：
- 品牌名：【】
- 品类：【】
- 创始人背景：【】
- 创立契机：【解决了什么问题/发现了什么机会】
- 品牌理念：【】
- 目标用户：【】
- 现阶段成绩：【销量/用户数/获奖等】

要求：
1. 300字版（用于详情页和社交媒体）
2. 100字版（用于名片和简介）
3. 一句话版（用于slogan）
4. 不要假大空，要有具体的故事细节
5. 语气：真诚、有温度，不要企业官话
```

## 邮件营销模板

**适用场景**：外贸开发信、客户维护邮件、活动通知邮件。

```
你是一个邮件营销专家，精通高打开率邮件的写作。

请帮我写一封【外贸开发信】：
- 我们的产品：【LED灯具，工厂直供】
- 目标客户：【欧洲中小型灯具经销商】
- 我们的优势：【MOQ低至50件、支持OEM、CE/ROHS认证齐全】
- 目标：【让对方回复邮件，建立初步联系】

要求：
1. 标题5个备选（打开率的关键）
2. 正文控制在150词以内（英文）
3. 第一句话就要吸引注意力（不要"I hope this email finds you well"）
4. 不要一上来就推销，先提供价值
5. 有明确的CTA（Call to Action），且门槛要低
6. 附上中文翻译版
```

## 用户证言/好评改写

**适用场景**：把真实好评整理成可用于营销的素材。

```
你是一个用户口碑营销专家。

以下是我们产品收到的真实好评，请帮我改写为可用于不同场景的营销素材：

原始好评：
- "用了三个月了，确实比以前那个好用，电池很耐用"
- "帮婆婆买的，她说很轻便，自己能推"
- "质量不错，就是颜色和图片有点色差，不过可以接受"

请改写为：
1. 详情页「用户说」版块文案（3条，每条50字以内，保留真实感）
2. 主图角标文案（1句话，10字以内，如"3个月真实好评"）
3. 朋友圈分享版（像用户自己发的，100字以内）

要求：
- 保留原始好评的真实语气
- 不要过度美化，有小缺点反而更真实
- 不要加感叹号和营销词
```
