## 一、它和普通 AI、和 Codex 有什么不一样

- **不是聊天框，是干活的**：它直接读你的文件、改文件、运行命令，全程用大白话指挥。
- **理解整个项目**：不是只看你贴的一段，而是能把整个文件夹的内容都"看在眼里"，跨文件改动。
- **能力强、但偏技术**：相比对小白更友好的 [Codex](/tools/codex-guide)，它更硬核，但上限更高。Anthropic 自己内部团队大量用它开发产品。

## 二、先搞懂"终端"——它是这一切的地基

后面装软件都要用到"终端"，先花一分钟认识它：

> **终端 = 电脑里一个黑色的输入框窗口，你在里面打字（敲命令）来指挥电脑。**

**怎么打开它：**

- **Windows（普通电脑）**：点开始菜单，搜索 **"PowerShell"** 或 **"命令提示符"**，点开，会弹出一个黑/蓝色窗口——这就是终端。
- **Mac（苹果电脑）**：按 `Command + 空格`，输入 **"终端"** 两个字，回车打开。

用法很简单：**把命令复制 → 粘贴进这个窗口 → 按回车**，就这么三步。下面所有"命令"都这么操作。

## 三、国内怎么用：四步安装

![Claude Code 安装四步](/images/guides/claude-code-install.svg)

::: tip 怕麻烦?有"一键脚本"
[claude-code-guide 中文指南](https://github.com/claude-code-chinese/claude-code-guide) 提供了**一键安装脚本**：把中转服务给你的"钥匙(API key)"和"地址"填进去，它能自动把下面这些步骤跑完。新手强烈建议用它省事。（打开是个英文页面，但里面的中文说明照着走即可；**实在不想碰英文页面，就照下面手动四步来，一样能装好。**）
:::

如果手动来，四步：

**第 1 步：装 Node.js（基础软件）**
打开官网 **nodejs.org**（**认准这个网址，别从山寨下载站下**），下载安装包，**像装微信一样双击、一路"下一步"**装好。它是 Claude Code 的"地基"。

**第 2 步：用命令安装 Claude Code**
打开终端（见上一节），把下面这行**整行复制粘贴进去，按回车**，等它装完：
```bash
npm install -g @anthropic-ai/claude-code
```

**第 3 步：验证装好没**
在终端里输入下面这行回车，如果能显示出一串版本号，就是装成功了：
```bash
claude --version
```

**第 4 步：填上"钥匙"和"地址"**
去你选的**国内中转服务**网站注册、充值，拿到两样东西：**API key（钥匙）** 和 **接入地址**。

填法说白了就是给电脑记两个值（你照着中转服务给的实际值替换即可）：
```bash
# 接入地址（中转服务会给你一个网址）
ANTHROPIC_BASE_URL=https://中转服务给你的地址/
# 你的钥匙（一长串字符，别外泄）
ANTHROPIC_AUTH_TOKEN=你的API_key
```
**这一步用前面说的"一键脚本"会自动帮你填好，新手不用手动操作。** 填好后，在终端输入 `claude` 回车，就进入 Claude Code 了。

::: warning 花钱 & 中转安不安全（老板必看）
- **大概花多少**：走中转一般是"先充值、按用量扣费"。轻度使用通常**一个月几十到一两百元量级**（大致量级，需验证）。建议**先小额充值（比如几十元）试水**，跑顺了再加。
- **怎么选中转、别被坑**：优先选有口碑、能开发票、客服在线的服务，别图便宜用来路不明的小站；你的"钥匙"别外泄，等于别人能花你的钱。**渠道更新快，本文不写死某一家**；拿不准选哪家，[加我微信](/about)帮你把关。
- **数据安全**：你的对话和代码会经过中转服务，**别用它处理特别敏感的商业机密**；重要资料先别让它碰。
:::

## 四、用好它的三条核心心法

![Claude Code 的正确姿势：先规划再动手](/images/guides/claude-code-flow.svg)

### 1. 先让它列计划，你点头它再动手（最重要）

别一上来就让它写。先用 **Plan 模式**让它把"打算怎么做"讲清楚，你确认方向对了，再放它去执行。活越复杂，这一步越值钱——能避免它朝错误方向猛干半天。

### 2. 管好"对话"，别让它记太多反而糊涂

![管好上下文：/compact 压缩、/clear 重置](/images/guides/claude-code-context.svg)

AI 的"注意力"有限，对话堆太长反而会变笨。**下面这些都是在它的输入框里直接打字输入的指令：**

- 一件事干完了，打 `/clear` 回车 → **清空重置**，开始下一件事
- 对话太长时，打 `/compact` 回车 → **把关键决定压成摘要**，留住重点
- 想让它看某个文件，打 `@` 再选文件 → 比你复制粘贴更准

### 3. 给项目放一份"说明书"：CLAUDE.md

在你项目最外层的文件夹里，**用记事本新建一个文本文件，文件名改成 `CLAUDE.md`**，里面写清楚：这个项目是干嘛的、有什么规矩、不要碰哪些东西。Claude Code **每次都会先读它**，相当于给新员工一份入职手册——交代一次，往后省心。

## 五、典型用法举例

- "帮我在这个网站加一个'联系我们'的表单" → 它读懂现有结构，自己找到该改的文件
- "这个页面在手机上显示错位了，修一下" → 它定位问题、改样式、让你看效果
- "把这个老项目升级一下，别改坏功能" → 跨文件批量处理，还能帮你验证

## 六、安全 & 搞砸了怎么办（老板必看）

- **新手务必开"每步先问你"**：可以把它设成执行关键操作前先征求你同意，你看到不对就别批准。
- **圈定地盘**：只让它在某个专门文件夹里干活，别让它满电脑乱动。
- **改坏能退回**：配合"存档/历史版本"，可回到上一个好版本。
- **钥匙看好**：API key 不要发给别人、不要截图外传。
