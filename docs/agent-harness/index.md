# Agent Harness 实战系列

> 2025 年是 Agent 元年，2026 年是 Agent Harness 元年。做 Demo 容易，上生产难——难的不是模型，是管理模型的那套系统。

## 这个系列讲什么

**Agent Harness（智能体框架）** 是让 AI Agent 在真实业务中稳定运行的管理系统。就像你招了一个聪明的新员工，光有能力不够，还得有工作手册、汇报流程、权限管理、出错处理方案。

这个系列用大白话讲清楚三件事：

| 篇目 | 核心问题 | 适合谁 |
|------|---------|--------|
| [为什么要懂](/agent-harness/why) | Agent 为什么上不了生产？Harness 解决什么问题？ | 所有想用 AI 的老板 |
| [怎么搭建](/agent-harness/how) | 5 种经过验证的搭建模式，从简单到复杂 | 想动手的技术负责人 |
| [核心组件](/agent-harness/what) | 一个完整的 Harness 包含哪 11 个部分？ | 要评估方案的决策者 |
