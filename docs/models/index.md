# 模型选择指南

> 面向电商老板和知识付费创业者，帮你快速找到最适合业务场景的 AI 模型。

AI 模型种类繁多，但你不需要全部了解。根据电商和知识付费的核心需求，我们将模型分为 **四大类**，每类推荐最实用的选择。

## 按内容类型选择

| 类型 | 推荐模型 | 典型场景 |
|------|----------|----------|
| [文字文案](/models/text) | Gemini 2.5 Pro / 豆包 | 商品文案、课程大纲、营销推文、客服话术 |
| [图片生成](/models/image) | Nano Banana (Google) | 商品主图、课程封面、社交媒体配图、海报 |
| [音频生成](/models/audio) | MiniMax 声音复刻 / Suno | 课程配音、品牌声音、背景音乐、有声内容 |
| [视频生成](/models/video) | Seedance 2.0 | 商品展示视频、课程宣传片、短视频内容 |

## 快速决策流程

```
你需要什么内容？
├── 文字 → Gemini 2.5 Pro（海外）/ 豆包（国内）
├── 图片 → Nano Banana
├── 音频
│   ├── 人声/配音 → MiniMax 声音复刻
│   └── 音乐 → Suno
└── 视频 → Seedance 2.0
```

## 成本参考

| 类型 | 免费额度 | 付费参考 |
|------|----------|----------|
| 文字 | 大部分有免费版 | API 调用约 ¥0.01-0.1/千字 |
| 图片 | 每日有限免费次数 | 约 ¥0.1-0.5/张 |
| 音频 | 试用额度 | 约 ¥0.01-0.05/秒 |
| 视频 | 试用额度 | 约 ¥0.5-2/秒 |

:::
