# Agent 搭建实战指南

> 概念看够了，动手吧。这个系列教你从零搭建第一个 Agent，测试它，然后上生产。

## 为什么需要这个指南

你现在可能已经看了很多关于 Agent 的概念和架构，但一到"那我怎么开始做"就卡住了。

这很正常。**从"我知道 Agent 是什么"到"我有一个能用的 Agent"之间，隔着一个巨大的鸿沟。** 这个指南就是帮你跨过去的桥。

```
对话 → 工具 → Skill → MCP → 【搭建 Agent】→ 系统运营
                                    ↑ 你在这里
```

你已经知道了 [什么是 Agent](/concepts/agents)、[怎么写 Skill](/agent-skills/index)、[怎么接 MCP](/mcp/index)。现在把这些东西组装起来，做一个真正能干活的 Agent。

## 选哪个平台

| 平台 | 适合谁 | 代码要求 | 中国可用 | 起步成本 |
|------|--------|---------|---------|---------|
| **扣子 (Coze)** | 不会代码的老板 | 零代码 | 原生 | 免费 |
| **Dify** | 有技术团队 | 低代码 | 可自部署 | 开源免费 |
| **Claude** | 开发者 | 写代码 | 需翻墙 | API 按量付费 |
| **n8n** | 想做工作流自动化 | 低代码 | 可自部署 | 开源免费 |

::: tip 选择建议
- 快速验证想法 → **扣子**（今天就能做出来）
- 数据要留在自己手里 → **Dify 自部署**
- 有开发团队、要深度定制 → **Claude API / OpenAI API**
- 要跟现有系统对接（邮件、表格、数据库）→ **n8n**
:::

从 [搭建你的第一个 Agent](/agent-guide/first-agent) 开始。
