# AI 编程框架与平台

> 这些是构建 AI 应用时最常用的框架和平台，了解它们能帮你做技术选型。

## 62. LangChain

最流行的 LLM 应用开发框架，提供了连接模型、管理提示词、串联工作流的标准化工具链。

## 63. LlamaIndex

专注于将你的私有数据与 LLM 连接的框架，特别擅长构建 RAG 系统和知识库问答应用。

## 64. Vercel AI SDK

Vercel 提供的 AI 应用开发工具包，让前端开发者可以轻松在 Web 应用中集成流式 AI 对话功能。

## 65. OpenAI API

OpenAI 提供的模型调用接口，通过它可以在自己的应用中使用 GPT、DALL·E 等 AI 能力。

## 66. Anthropic API

Anthropic 提供的 Claude 模型调用接口，以安全性和长上下文处理能力著称。

## 67. Function Calling（函数调用）

让 AI 模型能够调用外部工具和函数的能力，比如让 AI 查询数据库、发邮件、操控软件。

## 68. MCP（模型上下文协议，Model Context Protocol）

Anthropic 推出的开放标准协议，让 AI 模型能以统一方式连接和使用各种外部数据源和工具。

## 69. Hugging Face

全球最大的 AI 模型和数据集共享平台，提供数十万个开源模型供开发者免费使用和部署。

## 70. 低代码/无代码 AI（Low-Code / No-Code AI）

通过拖拽组件和可视化配置就能构建 AI 应用的平台，大幅降低技术门槛。

## 71. Dify

开源的 LLM 应用开发平台，提供可视化的工作流编排，让非技术人员也能构建 AI 应用。

## 72. Coze（扣子）

字节跳动推出的 AI 应用开发平台，支持可视化搭建 AI Bot 并发布到多个渠道。

## 73. Streamlit

用 Python 快速构建数据应用和 AI 演示界面的框架，几行代码就能创建交互式 Web 界面。

## 74. Next.js

基于 React 的全栈 Web 开发框架，是目前构建 AI 驱动 Web 应用最主流的前端框架之一。
