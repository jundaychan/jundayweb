# 热门 Skills 推荐：老板最该用的 AI 能力

> 别从零开始造轮子。这些是全网最火的 AI Skills / GPTs / Bot，按你实际干的活分类，拿来就用。

## 怎么用这个推荐列表

1. **找到你最痛的场景**（下面按场景分类）
2. **优先选中国能直接用的**（标了"需翻墙"的要科学上网）
3. **先试免费的**，效果好再升级付费
4. **不要贪多**，先把一个场景跑通

## 怎么选平台：一张图说清楚

```
你在中国，不想翻墙？
  ├── 要零代码 → 扣子（Coze）
  ├── 要数据自主可控 → Dify 自部署
  ├── 用飞书 → 飞书多维表格
  └── 用钉钉 → 钉钉宜搭

你能翻墙？
  ├── 想用现成的 → GPT Store
  ├── 想做自动化 → n8n
  └── 想做多平台 → Dify + n8n
```

## 三个行动建议

### 1. 今天就做

打开[扣子商店](https://www.coze.cn)，搜索你最需要的场景（小红书文案 / 电商客服 / 数据分析），找一个评分高的 Bot 试用。**10 分钟就能感受到效果。**

### 2. 这周做

从上面的 GitHub 开源项目里，找到适合你场景的提示词或模板，在你日常用的 AI 工具里试试。推荐从 [awesome-chatgpt-prompts](https://github.com/f/awesome-chatgpt-prompts) 开始。

### 3. 这个月做

选一个高频场景（比如客服、文案），用 Dify 或扣子搭建一个完整的 AI 工作流，让它自动运行。详见 → [搭建第一个 Agent](/agent-guide/first-agent)
