## 豆包 (Doubao)

**出品方**：字节跳动（抖音母公司）

### 价格

- **对话版**：基本免费使用
- **API 价格**：每百万 token 输入 **0.15 元人民币**，输出 1.5 元——这个价格比 ChatGPT 便宜约 **99%**
- 2025 年字节跳动将 API 价格下调了 70%

### 核心能力

- 中文处理能力极强（字节系产品，针对中文优化）
- Doubao-1.5-Pro 在多项基准测试中超过 GPT-4o 和 Claude 3.5
- 支持文字、图片、语音多模态
- 与抖音生态深度整合

### 用户规模

- **1.59 亿月活用户**，全球使用量最大的 AI 助手之一

### 适合谁

- 每天需要写大量文案的电商运营
- 需要快速出方案、出回复的客服团队
- 预算有限但需要高频使用 AI 的中小企业

### 注意事项

- 台湾第三方安全评估发现豆包在数据收集方面存在隐私顾虑（见[常见问题-数据安全](/faq#ai-会不会泄露我的商业数据)）
- 敏感商业数据建议谨慎使用

> 数据来源：字节跳动官方、各科技媒体评测

## 通义千问 (Qwen)

**出品方**：阿里云

### 价格

- **对话版**：完全免费，直接使用 Qwen-Max 最强模型
- **API 价格**：Qwen 2.5 72B 每百万 token 约 **1.6 元人民币**（$0.23），约为 GPT-4o 的 **1/10**
- 支持从 $10 到 $5,000 的预购优惠

### 核心能力

- Qwen3 模型支持"思考模式"和"快速模式"切换，可以灵活控制成本和质量
- 支持 119 种语言
- 上下文窗口最长 **12.8 万 token**
- Qwen 2.5-Omni：支持文字、图片、视频、音频输入，同时输出文字和语音
- 开源模型，企业可以私有化部署

### 适合谁

- 需要免费高质量对话的个人用户
- 需要 API 接入自有系统的技术团队
- 对数据安全有要求、想私有化部署的企业（开源模型可本地运行）

### 优势

- **免费对话 + 低价 API + 开源可私有部署**，这个组合在国内没有对手
- 阿里云生态整合，适合已经在用阿里云的企业

> 数据来源：阿里云官网、Qwen GitHub 仓库

## Kimi

**出品方**：月之暗面（Moonshot AI）

### 价格

- **免费额度**：每月 100 万 token
- **付费方案**：5.2 元（4天体验）到 399 元（年度优先访问），共 6 个档位
- **API 价格**：输入 $0.004/千 token，输出 $0.006/千 token

### 核心能力

- **超长上下文：25.6 万 token**——可以一次性分析一本书、一份完整合同、几十页的财报
- Kimi K2：万亿参数 MoE 模型，开源，性能对标 GPT-4
- 在代码和推理任务上表现突出

### 用户规模

- **3600 万月活用户**
- 2026 年被认为是中国最快成长的 AI 独角兽

### 适合谁

- 需要分析长文档的老板（合同审核、财报分析、市场报告）
- 需要深度长文输出的内容创作者
- 重视推理和逻辑能力的决策场景

### 与豆包/通义的区别

Kimi 的核心差异化是**超长上下文**。如果你的工作经常涉及长文档处理（几万字的材料分析、多份文件对比），Kimi 是目前最好的选择。短文案和日常问答用豆包更划算。

> 数据来源：月之暗面官网、科技媒体报道

## 文心一言 (ERNIE Bot)

**出品方**：百度

### 价格

- **2025年4月1日起，个人用户完全免费**（含桌面版和移动版）
- 之前的付费方案：49.9 元/月（文心4.0）、99 元/月（联合会员）
- **API 价格**：每百万 token 输入约 $0.55 起

### 核心能力

- 深度搜索 + 增强推理
- 与百度搜索生态整合，联网能力强
- ERNIE-4.0 Max 可与 Claude 和 GPT-4o 竞争
- 支持外部工具调用

### 适合谁

- 需要 AI 帮忙搜索最新信息的场景（市场调研、竞品分析）
- 已经在用百度生态（百度统计、百度推广）的企业
- 预算为零、只想免费用的个人用户

### 注意事项

- 免费之后质量是否会下降有待观察
- 在创意写作和长文方面不如 Kimi

> 数据来源：百度官网、各科技媒体报道

## ChatGPT

**出品方**：OpenAI

### 价格

- **免费版**：有限制
- **Plus**：$20/月（约 145 元人民币）
- **Pro**：$200/月

### 中国可用性

::: danger 中国大陆无法直接访问
- 需要 VPN 翻墙才能使用
- 使用 VPN 访问**违反 OpenAI 服务条款**，有封号风险
- 不推荐作为中国企业的主力工具
:::

### 如果你能访问的话

- GPT-4o 仍然是综合能力最强的模型之一
- 英文场景下明显优于国内模型
- 插件生态丰富（联网、代码执行、图片生成一体化）
- 适合有海外业务的企业

### 建议

如果你的业务主要面向中国市场，用国内工具就够了。如果有跨境电商、外贸业务需要处理大量英文内容，ChatGPT 仍然值得付费使用。

> 数据来源：OpenAI 官网

## Claude

**出品方**：Anthropic

### 价格

- **免费版**：有限制
- **Pro**：$20/月

### 中国可用性

::: danger 严格限制中国用户
2025年9月，Anthropic 实施了严格的地区限制：
- 明确禁止中国控股超过 50% 的企业使用
- 影响了依赖 Claude 的多个工具（包括字节跳动的 Trae）
- 中国用户/企业**不受官方支持**
:::

### 如果你能访问的话

- 长文写作能力强，输出质量稳定
- 代码能力突出
- 20 万 token 上下文窗口

### 建议

对中国企业来说，Claude 基本不可用。类似能力可以用 Kimi（长文）或通义千问 API（代码和推理）替代。

> 数据来源：Anthropic 官网、科技媒体报道

## 实测建议

### 如果你只想试一个

先用**豆包**。免费、中文好、速度快，能满足 80% 的日常需求。

### 如果你愿意多试几个

1. **豆包**做日常文案和快速问答
2. **Kimi**做长文档分析和深度报告
3. **通义千问**做需要最新信息的搜索和调研

这三个配合使用，基本覆盖所有文字类 AI 需求。总成本：**零元**（都有免费额度）。
