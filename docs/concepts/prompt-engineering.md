# 提示工程（Prompt Engineering）

## 什么是提示工程

提示工程就是**学会怎么跟 AI 说话**。同样的 AI 工具，会问问题的人能得到远超不会问的人的结果。

这不是编程，不需要技术背景——更像是一种**沟通技巧**。

## 为什么重要

看两个例子的对比：

::: info 差的提示
"帮我写个营销方案"
:::

::: tip 好的提示
"我是一家做儿童编程教育的公司，目标客户是一线城市 30-40 岁的家长。请帮我写一个暑假招生的营销方案，包括：1）核心卖点提炼 2）3 个推广渠道建议 3）每个渠道的具体执行步骤。预算在 5 万元以内。"
:::

第二个提示给了 AI 足够的**上下文**，所以能得到更有针对性的回答。

## 实用技巧

### 1. 给清楚背景

告诉 AI 你是谁、你在做什么、目标是什么。

### 2. 明确输出格式

告诉 AI 你想要什么格式的回答：表格、列表、分步骤、字数限制等。

### 3. 给个角色

> "你是一个有 10 年经验的市场总监，请从专业角度帮我分析..."

让 AI 扮演特定角色，可以得到更专业的回答。

### 4. 分步骤来

复杂任务不要一次性丢给 AI，拆分成多个小步骤：

1. 先让 AI 了解背景
2. 再让它提出方案
3. 然后让它细化你最感兴趣的部分
4. 最后让它检查和优化

### 5. 让 AI 先想再答

> "在回答之前，请先列出你需要考虑的关键因素，然后再给出建议。"

这种"思维链"方式能让 AI 给出更深思熟虑的回答。

## 企业常用提示模板

### 会议纪要
```
请将以下会议录音转录整理成会议纪要，包含：
1. 会议基本信息（时间、参与人）
2. 讨论要点（分条列出）
3. 决议事项
4. 待办事项（标注负责人和截止时间）

录音内容：[粘贴内容]
```

### 竞品分析
```
请帮我分析 [竞品名称] 这家公司：
1. 核心产品和服务
2. 目标客户群
3. 商业模式
4. 优势和劣势
5. 对我们的启示（我们是做 [你的业务] 的）
```

### 邮件撰写
```
帮我写一封邮件给 [对象]，目的是 [目的]。
语气要求：[正式/友好/简洁]
关键信息：[列出要点]
字数控制在 200 字以内。
```
